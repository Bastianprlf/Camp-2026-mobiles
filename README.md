# Camp Guide 2026 - Apprentissage Morse et Chants (Version mobiles)

## Description

Ce site a été créé pour apprendre le morse et les chants pour le camp guide 2026.

Il a pour objectif d’aider les participantes et participants à :

- Apprendre et pratiquer le code Morse  
- Découvrir et mémoriser les chants du camp  
- Se préparer pour le Camp Guide 2026 de manière ludique et interactive  

## Objectif du projet

Ce projet sert de support d’apprentissage pour faciliter la préparation au camp guide.

## Contenu du site

- Leçons et exercices de Morse  
- Répertoire des chants du camp  
- Outils de pratique et de révision  

## Camp Guide 2026

Prépare-toi, apprends et chante pour être prêt(e) pour l’aventure !
